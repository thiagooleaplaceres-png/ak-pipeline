01
Adding a Property
Click + Add Property. Start typing any NYC street number or name — suggestions appear instantly from just 2 characters. Select the correct address from the dropdown (it fills automatically), add Apt/Unit if needed, choose Sale or Rent, set Listed or Closed, and select all agents involved. Press Save Property to sync for the whole team.
02
Marketing Materials
Each property has two columns: Just Listed tracks 4 materials — Building Postcard, Neighborhood Postcard, 4-Page Brochure, and Window Scan. Closed/Rented tracks 2 postcards only. Check any box to activate that material for tracking. Unchecked items are not counted in progress.
03
Advancing Stages
Click Start → to begin a stage, then Next → to advance it through each pipeline step. Each step lights up in its assigned color. Use ← Back to correct a mistake. The current step name is displayed in color below the visual pipeline.
04
Listed vs. Closed / Rented
Just Listed (green column) tracks all 4 marketing materials for active listings. Closed/Rented (red column) tracks only the two postcards — one for your Building, one for your Neighborhood. Both pipelines run completely independently.
05
Real-Time Sync
Every action — adding a property, checking a box, advancing a stage — saves instantly and appears for all team members simultaneously. The green Live dot in the nav confirms your connection. A ↻ Saved toast confirms each update. No refreshing needed.
06
Removing a Property
Click the Remove Property button on any card at any time — no completion required. A confirmation prompt will appear. Once confirmed, the property and all its data are permanently deleted from the shared database for the entire team.
