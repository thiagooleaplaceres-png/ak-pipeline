Property Pipeline

Real-time
Team Reference
01
Adding a Property
Click + Add Property in the pipeline section. Fill in the street address using the NYC autocomplete, add the Apt # if applicable, select Sale or Rent, choose Listed or Closed, and select all agents involved. Click Save Property — it appears for the entire team instantly.
02
Marketing Materials
Each property has 4 marketing material checkboxes. Check the box to activate a material for tracking. Each material has its own pipeline: Requested → Anna Review → Anna Approved → Marketing Approved → Ordered/Delivered. Postcards have an extra Ordered and Delivered step.
03
Advancing Stages
Once a material is checked, click Next → to advance it through each pipeline step. The visual pipeline shows colored nodes for each step. Click ← Back if a correction is needed. The current stage label is shown in color below the pipeline.
04
Listed vs. Closed
Each material is split into two columns: Listed (green) and Closed (red). Track them independently — a building postcard for the listing and a separate one for when it closes. This keeps the two processes clearly separated.
05
Real-Time Sync
Every change — adding a property, advancing a stage, updating agents — is saved instantly and visible to all team members. The green Live dot in the nav confirms your connection. A sync toast appears confirming each save.
06
Removing a Property
Properties can only be removed once all activated stages are marked complete. This ensures nothing gets deleted prematurely. The Remove button becomes available automatically when all stages reach their final step.
